# measurement_app MVP 功能列表

## 版本資訊
- 版本：v0.1 MVP
- 日期：2026-05-14
- 目標：驗證釣魚族群使用意願與分享傳播性

---

## 核心功能

### 1. 照片上傳與拍攝
| 功能 | 說明 | 優先級 |
|------|------|--------|
| 相機拍攝 | 開啟相機，即時拍攝 | P0 |
| 相簿選取 | 從相簿選擇現有照片 | P0 |
| 拍照引導 | 提示用戶將魚與測量墊/參考物放在同平面 | P1 |
| 自動檢查 | 檢測畫面中是否有測量墊/參考物 | P1 |

### 2. AI 魚體偵測
| 功能 | 說明 | 優先級 |
|------|------|--------|
| 魚輪廓偵測 | 自動框選魚體範圍 | P0 |
| 魚頭/魚尾定位 | 標記魚頭前端與尾鰭末端 | P0 |
| 彎曲魚身處理 | 用曲線/骨架計算弧長（不只直線） | P1 |
| 手動調整 | 允許用戶微調自動框選的點 | P1 |

### 3. 尺度校正
| 功能 | 說明 | 優先級 |
|------|------|--------|
| 參考物偵測 | 自動辨識畫面中的測量墊/刻度板 | P0 |
| Pixel-to-cm 換算 | 根據參考物計算像素比例 | P0 |
| 多參考物支援 | 支援不同尺寸的測量墊/硬幣/手機 | P2 |
| 誤差提示 | 當角度過大或參考物不清時提示重拍 | P1 |

### 4. 長度計算與顯示
| 功能 | 說明 | 優先級 |
|------|------|--------|
| 自動計算長度 | 顯示魚的全長（cm） | P0 |
| 尺標疊加 | 在照片下方或旁邊顯示視覺化尺標 | P0 |
| 長度記錄 | 儲存本次量測結果 | P0 |
| 單位切換 | 支援 cm / inch | P2 |

### 5. 戰績卡生成
| 功能 | 說明 | 優先級 |
|------|------|--------|
| 資訊填寫 | 魚種（手動輸入或選擇） | P0 |
| | 日期（自動帶入） | P0 |
| | 地點（GPS 或手動輸入） | P1 |
| | 釣法 / 餌料（手動輸入） | P1 |
| 戰績卡排版 | 自動生成美觀分享圖片 | P0 |
| 模板選擇 | 提供 2-3 種戰績卡風格 | P2 |

### 6. 分享功能
| 功能 | 說明 | 優先級 |
|------|------|--------|
| 儲存到相簿 | 儲存戰績卡到手機 | P0 |
| 分享到社群 | Instagram / Facebook / LINE 等 | P0 |
| 複製連結 | 產生戰績卡圖片連結（如後端支援） | P2 |

### 7. 歷史紀錄
| 功能 | 說明 | 優先級 |
|------|------|--------|
| 紀錄列表 | 顯示過往量測記錄 | P1 |
| 紀錄詳情 | 查看單筆詳細資訊 | P1 |
| 刪除紀錄 | 刪除不需要的記錄 | P2 |

## 技術需求

### 前端
- 手機 App（優先做 iOS，Android 第二階段）
- 相機權限與相簿存取
- 基本圖片處理與疊加渲染

### 後端 / AI
- 魚體偵測模型（YOLO / Detectron / SAM 等）
- 參考物偵測與尺度校正邏輯
- 長度計算演算法（直線 + 曲線）
- 戰績卡圖片生成 API

### 資料儲存
- 本地端：量測記錄暫存
- 雲端（第二階段）：用戶帳號、歷史紀錄同步、社群分享連結

---

## MVP 不做的事（第二階段再做）

- ❌ AR 即時量測（LiDAR / ARKit / ARCore）
- ❌ 魚種 AI 自動辨識（先用手動輸入）
- ❌ 用戶帳號系統（先做匿名本地使用）
- ❌ 社群功能（按讚、追蹤、排行榜）
- ❌ 比賽活動系統
- ❌ 漁獲資料庫統計

---

## 驗證目標

1. **釣魚族會不會用** — 觀察用戶是否願意拍攝含測量墊的照片
2. **願不願意配合拍照規則** — 追蹤重拍率與放棄率
3. **分享卡有沒有傳播性** — 追蹤戰績卡分享次數與平台

---

## 開發順序建議

**第一階段（2-3 週）**
1. 照片上傳 + 基本 UI
2. 手動標記魚頭/魚尾（先不做 AI 偵測）
3. 手動輸入參考物尺寸（先不做自動偵測）
4. 長度計算 + 尺標顯示
5. 戰績卡生成（簡單模板）

**第二階段（2-3 週）**
6. 接入 AI 魚體偵測模型
7. 自動參考物偵測
8. 手動調整框選點
9. 歷史紀錄功能

**第三階段（後續）**
10. 魚種 AI 辨識
11. 用戶帳號與雲端同步
12. 社群功能
13. AR 即時量測 Pro 版
